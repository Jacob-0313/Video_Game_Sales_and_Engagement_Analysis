# 🎮 Video Game Sales & Engagement Analysis

## 📊 Project Overview

This project analyzes **video game sales trends, player engagement, and market performance** using data analytics and visualization techniques.

The goal is to discover insights such as:

* Top selling games
* Platform performance
* Genre popularity
* Regional sales trends
* Market growth patterns

The analysis helps understand **what drives video game success in the global gaming industry**.

---

## 🚀 Technologies Used

🐍 Python
📊 Pandas
📈 Matplotlib
📉 Seaborn
📓 Jupyter Notebook

---

## 📂 Dataset

The dataset includes information about:

* Game titles
* Platforms
* Genres
* Publishers
* Global sales
* Regional sales (NA, EU, JP, Other)
* Release year

Source files included in the repository:

```
vgsales.csv
games.csv
R_Game_Data.csv
R_Sale_Data.csv
```
